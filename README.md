# Repo for Bradley Rodgers portfolio site

This is my portfolio site which can be viewed at [http://www.bradleyrodgers.github.io](http://www.bradleyrodgers.github.io).

All instructions require you to be in the relevant directory.  So in terminal:
`cd path/to/local/bradleyrodgers.github.io`

## Initial set up
1. `npm install`
2. `npm start`
3. open index file in a browser


## Development

1. `npm run watch`
2. open index file in a browser
